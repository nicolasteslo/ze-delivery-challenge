## Run the project locally

To clone the project using Github CLI:

```sh
gh repo clone nicolasteslo/ze-delivery-challenge
```

To install dependencies:

```sh
cd ze-delivery-challenge && yarn
```

To run the project

```sh
yarn dev
```
